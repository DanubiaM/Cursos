Como adicionar imagens em seu site?

Você devera usar a tag 'img', ela não possui fechamento de tag /img.
Sempre que for utilizar uma imagem com muitas cores use o formato JPEG
Para imagens com poucas cores use GIF ou PNG

src
usamos o src para adicionar o link da imagem.
alt
usamos o alt para descrever a imagem.
title
usamos o title para adicionar um titulo a imagem.
height
usamos o height para definir a altura da imagem.
width
usamos p width para definir a largura da imagem.
Por fim a maneira de adicionar uma imagem é 'img="link" alt="descrição" title="Titulo" '
A descrição e o titulo da imagem serve apenas para facilitar a busca da imagem por um mecanismo de pesquisa (Google). Porem você pode realmente adicionar uma descrição na imagem utilizando figuras
Utilize "figure" coloque a imagem dentro da tag e adicione outra tag chamada "figcaption" para descrição
