Cores

As cores que vemos na tela são formadas por uma mistura de vermelho, verde e azul. No CSS existe três tipos de cores possiveis para ser adicionado como valor, RGB, HEX, Name.

RGB

É no formato rgb(0,0,0) sendo os três valores possiveis variando entre 0 a 255. A primeira coordenada é vermelho, segunda verde e a ultima azul.

HEX

As cores no sistema hexadecimal deve ser do seguinte estilo: #324bb1.

Color Name

Existe 147 tipos de cores posssiveis, basta apenas adicionar o nome da cor.

Termos

Hue

Representa as cores em um circulo, de o a 360 graus.

Saturatição

Referece a quantidade de cinza na cor. Uma saturação máxima não contem cinza, já a saturação minima a cor é cinza.

Brightness

Referece ao quão preto é a cor. No máximo não existe preto, no mínimo é preto.

Opacidade

A oracidade de um objeto é visto no CSS3 com o estilo de cor RBGA onde temos uma quarta coordenada sendo a opaciade. Esse valor de opaciade pode variar de 0 à 1, sendo 0.5 o equivalente a 50%...

HSL e HSLA

O HSL é a junção do Hue, Saturatição e Brightness para formar uma cor. Logo é escrita da seguinte forma hsl(0,0%,78%), sendo o Hue o primeiro campo o valor da cor, o segundo campo a saturação em porcentagem e o terceiro campo a porcentagem do Brightness.
Assim como no RGBA, o HSLA utiliza uma coordenada para inserção da transparencia (variando de 0 a 1), sendo da seguinte maneira: HSLA(3,5%, 79%, 0.5)
