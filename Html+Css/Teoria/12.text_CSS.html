Controle de Texto

Pode ser dividido em dois grupos:
Aqueles que afeta diretamente a fonte e sua aparencia(typeface, bold, italic, size)
E aqueles que teria algum efeito no texto, independente da fonte ( cor, espaçamento entre as letras e palavras)

Terminologia Typeface

As seguintes famílias de fontes são comumentes usadas para as seguintes situações:
Serif: É usada em passagens longas por serem faceis de ler. Exemplo: Georgia, Times, Times New Roman
Sans-Serif: É usada para textos pequenos. Exemplos: Arial, Verdana e Helvetica .
Monospace: Usada em codigos por ser mais alinhada. Exemplos: Courier e Courier New
Alguns outros tipos de familia de fonte utilizadas são:
Cursive: Caracterizada por parecer ser escrita a mão. Exemplos: Comimc Sans MS e Monotype Corsive
Fantasy: São fontes descorativas, frequentemente usadas em titulos. Exemplos: Impact e Haettenschweiler

Alguns tipos de caracteristicas da fonte:

Weight
font-weight: tem os possiveis atributos lighter,bold Além desses possiveis valores, podesse adicionar uma escala de 100 à 900. Essa escala condiz com o quão negrito sera.
Style
Diz a respeito do seus aspecto cursiva:
font-style: tem os possiveis atributos normal, italic e oblique.
Stretch
font-stretch: tem os possiveis atributos condensed, regular e extended.

Font-Size

As propiedades da fonte size é para delimitar o tamanho da fonte. Pode ser especificada por pixels, porcentagens e ems.

Pixels
Os pixeis são usados por ter um controle preciso do controle de espaço do texto,o tamanho padrão no navegador é 16px. Exemplo: font-size= 300px.
Porcentagens
O tamanho normal do navegador é 75%. Exemplo: font-size= 30%
Ems
Equivale ao tamanho a largura de uma letra m. Exemplo font-size=1.2em

Tipo de escalas

Como ja apresentado, existe três tipos de escala, pixels, porcentagens e ems. Entretando existe um padrão a ser usado por elas. O padrão de tamanho de texto no navegador é 16px, se for usar % ou ems deve calcular o tamanho do texto baseado no tamanho padrão usado no navegador. Se usar 12px no corpo do texto, deve-se então usar 24px para títulos.

Font-face

Para usarmos fontes que não estão baixadas no computador, devemos usar o font-face. Ela é composta pelas seguintes propiedades: font-family(nome da fonte), src(o caminho da fonte) e format(especificado o formato).
Sites de fontes free: https://www.fontsquirrel.com/ e http://www.fontex.org/.
Existe varios tipos de formatos de fontes aceitos pelos navegadores como: eot, woff,ttf/otf e svg. Dessa mananeira, deve-se especificar diferentes caminhos dentro do src, para que possa utilizar varios de formatos de fonte, e aparecem em diferentes tipos de navegadores.
Use o site https://www.fontsquirrel.com/tools/webfont-gerator para gerar a fonte nos formatos anteriores(woff).

font-weight

Os tipos de atributos possiveis para font weight é: normal, bold,

font-style

A font-style nos permite adicionar o negrito dentro do CSS. Os possiveis valores é: normal, italique e oblique

text-transform

Podemos transformar uma fonte em maisucula utilizando a propiedade text-transform. Ela pode conter os seguintes valores: uppercase ,lowercase e capitalize .

text-decoration

Para poder decorar o texto existe o text-decoration. Ele possui os seguintes valores de atribuos: none, underline, overline line-through, blink

Espaçamento das linhas

Para fazermos um espaçamento das linhas devemos utilizar a propiedade line-height, essa propiedade vai alterar a altura da linha. Então nela deve conter os valores em numeros como em. Um bom valor é 1.4 em ou 1.5 em.

Espaçamento entre letras e palavras

Para fazer espaçamento entre as letras basta que use a propiedade letter-spacing e o valor em "em".
Já para fazer um espaçamento de palavras use-se word-spacing. Normalmente o espaçamento entre as palavras é de 0.25em

Alinhamento de texto

Você pode fazer o alinhamento do texto no CSS, para isso basta usar a propiedade text-align. Ela possui os seguintes valores possiveis: left, right, center, justify. O mais utilizado é o alinhamento left.

Alinhamento Vertical

O vertical-align é utilizado para realizar o alinhamento vertical de tags como img, em e strong. Além disso, pode-ser usado para fazer alinhamentos de células de uma tabela também. Ele não deve ser usado para fazer alinhamento de div e p.
Seus possiveis valores são: baseline, sub, super, top, text-top, middle, bottom e text-bottom. Além desses valores, também pode-se definir o tamanho do deslocamento vertical usando px ou %.

Identamento de Texto

text-indent

O identamento de texto é uma propiedade a ser colocada na primeira linha do paragrafo.
Normalmente o deslocamento é feito em px, sendo positivo para a direita, e caso deseje um deslocamento para a esquerda, basta fazer um deslocammento negativo.

Sombra Projetada

text-shadow

Podemos fazer um uma sobra nos textos que escrevemos, entretando alguns navegadores não suportam essa tecnologia.
Existe quatro valores a serem especificados no texto-shadow.

  1. O primeiro valor a ser especificado indica o quo longe deve ir a sombra, para direita ou esquerda, sendo especificado em px.
  2. O segundo valor indica a distancia que a sombra deve ir para cima ou para baixo. Também especificado em px
  3. O terceiro valor é opcional, serve para dizer o desfoque da sombra. Também dado em px.
  4. O ultimo valor é a cor da sombra.

Primeira Letra ou linha

:first-letter e :fisrt-line

Agora não trataremos mais sobre as propiedades e sim sobre pseudo-elementos. Podemos especificar um pseudo-elemento dentro do seletor "h1 :first-letter{ propriedades}".
A :first-letter serve para realizar a ação na primeira letra, já o first-line a ação é feita na primeira linha.

Pseudo-classes

Estilo de Links

:link e :visited

Esses elementos são chamados de pseudo-classes. Quando se trata de interação entre links usamos o :link e :visited. O :link apresenta estilos para links que não foram visitados. Já o :visited apresenta estilos para sites que ja foram visitados.

Respondendo aos usuarios

:hover, :active e :focus

Esse tipo de pseudo-classe é feito para realizar interação com o usuario.
O :hover aplicado quando o usuario passa por cima com o mouse de um elemento.
O :active a ação é realizada quando o usuario clica em cima do elemento.
O :focus é usado quando deseja dar um foco em um elemento que o usuario esta usando, como uma caixa de entrada. Dessa maneira ira realizar uma ação no elemento que o usuario esta usando.
