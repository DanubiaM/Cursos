Transição

A propriedade transition permite realizar mudança nos valores do box. Para isso é necessário definir as propriedades, e depois criar um :hover para a mudança poder realizar as mudanças com o transition.
Existe 4 tipos de propriedades de transição:
transition-delay
transition-duration
transition-property
transition-timing-function.

transition-timing-function

Essa propriedade especifica a velocidade em que desejamos que aconteça a transição. Possui os seguintes valores:

  1. ease: inicio rápido; meio lento; fim rápido
  2. linear: mesma velocidade do inicio ao fim
  3. ease-in: inicio lento; resto normal
  4. ease-out: final lento; resto normal
  5. ease-in-out: inicio e fim lento; meio normal

transition-duration

A propriedade transition-duration nos permite definir o tempo da transição. Normalmente definida em segundos, exemplo: 2s. Se colocada dentro do seletor hover, a transição do hover que ira durar 2s.

transition-property

Essa propriedade nos permite especificar que propriedade do seletor desejamos suavizar a transição. Dessa maneira, basta especificar: transition-property:width, que a transição do width sera suavizada.

transition-delay

Essa propriedade da um atraso para a transição da propriedade, também é especificada em segundos. Exemplo: transition-delay:1s.

transition resumido

Vimos o uso de cada propriedade de transição separadamente, mas assim como outras propriedades do CSS, também podemos utilizar a forma resumida. Para isso basta utilizar a propriedade transition:property duration function delay, property duration function delay ... Dessa maneira podemos adicionar quantas transições quisermos, basta apenas separar por virgula.

CurrenteColor

Essa propriedade permite que uma cor de uma propriedade seja repitida sem que seja necessário escrever o nome da cor novamente.
