Estilizando os marcadores da lista

list-style-type

Utilizamos essa propriedade para mudar os marcadores das listas, seja ela ul, ol ou li. Para listas não ordernadas usamos as seguintes propriedades:

  • none: nenhum marcador
  • disc: marcador em bolinha
  • circle: marcador em circulo
  • square: marcador em um quadrado preto

Para listas ordernadas, temos as seguintes propriedades:
  1. decimal: 1, 2, 3...
  2. decimal: 01, 02, 03...
  3. lower-alpha: a, b, c ...
  4. upper-alpha: A, B, C ...
  5. lower-roman: i, ii, iii...
  6. upper-roman: I, II, III...

list-style-image

É possivel criar marcadores utilizando imagens usando a propriedade list-style-image. Para isso basta que adicionamos o valor url() e dentro dele o caminho da imagem. Pode ser utilizado em listas do tipo ul e ol.

list-style-position

A posição do marcador em uma lista pode ser dentro do texto ou fora, sendo assim, existe dois possiveis valores para essa propriedade, o outside e inside. A outside é por padrão já feita automaticamente ao criar uma lista, ou seja é a o marcador do lado de fora. Já o inside adiciona o marcador junto ao texto, como se fizesse parte da linha.

border-spacing & border-collapse

Podemos adicionar ou remover um espaçamento entre as células de uma tabela utilziando as propriedades border-spacing e border-collapse. Por padrão, ao criar uma tabela, existe um pequeno vão entre as células, porem podemos aumentar ou diminuir este vão. Para aumentar utiliza os o border-spacing, sendo especificado o valor do vao em pixels. Já para retirar o vão utiliza-se o border-collapse, definindo seu valor como collapse;
