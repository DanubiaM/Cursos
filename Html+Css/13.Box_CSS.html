Box

Box Dimensões

Um box deve conter o tamanho apenas para caber o conteudo. Podendo conter uma largura(width) e uma altura(height)
Para especificarmos o tamanho de um box, usamos pixeis, porcentagem ou ems.
Quando usamos porcentagem o tamanho do box deve ser relativo ao tamanho da janela do navegador ou de outro box.
Se usado ems o tamanho do box é baseado no tamanho do texto dentro dele.

Limitando a Largura

min-width e max-width

  • A propiedade min-width é responsavel por especificar o menor tamanho que o box pode ter quando a janela do navegador é pequena.
  • Já a propiedade max-width indica a largura máxima que um box pode ter quando a janela do navegador esta no máximo.
Tente dimensionar o navegador!

min-height e max-height

De maneira similar ao método anterior, podemos utilizar o min-height e max-height para delimitar a altura do box.

overflow

Para resolver o problema da sessão anterior onde ocorre transbordo de conteudo do box usa-se a propiedade overflow. Ela possui dois valores hidden e scroll. A primeira ira esconder qualquer conteudo extra que não cabe no box.
A segunda ira fazer um box para realizar uma rolagem do conteudo, ou seja, ira utilizar uma barra de rolagem.

Borda, margem e preenchimento

Todo o box possui as três propiedades para alterar sua aparencia: border, margin e padding.

Border
Todo box possui uma borda, caso não especificado sera 0px. Uma borda separa beira de uma box da outra.
Margin
As margens ficam foram da beira da borda. É utilizado para criar um vão entre dois box.
Padding
O preenchimento é o espaço entre a borda do box e o conteudo dentro dele.

border-width

A largura de uma borda possui três tipos de valores: thin(fino), medium, thick(grosso). Além desses valores, pode espeficar sua largura usando pixels. NÃO É POSSIVEL UTILIZAR border-width SOZINHO, É NECESSÁRIO UTILIZAR O BORDER-STYLE JUNTO.
Para controlar a borda de cada vertice existe duas maneiras:

  1. Usando: border-top-width, border-right-width, border-button-width, border-left-width
  2. Ou usando: boerder-widht: 3px 1px 4px 5px

border-style

Podemos controlar o estilo de uma borda utilizando p border-style. Ele pode conter os seguintes valores:
Solid: uma linha sólida.
dotted: uma série de pontos com o espaçamento de 2px.
dashed: uma série de linhas curtas.
double: duas linahs solidas.
groove: parece ser esculpido na pagina.
ridge: parece se destacar na pagina.
inset: parece estar embutido na pagina
outset: parece que esta fora da tela.
hidden/none:não possui borda.
Pode-se colocar estilo em vertices separadas nas bordas, usando o border-possição-style

border-color

Podemos adicionar cores nas bordas também, tanto na borda toda ou em cada vértice da borda de maneira similar aos métodos anteriores. border-posição-color ou border-color: cor cor cor cor.

Padding

A padding como apresentado anteriormente, é o espaçamento entre o conteudo e as bordas. Assim como as propiedades das bordas, pode-se colocar o valor das padding de duas maneiras diferentes:
padding-posição, tal que posição= width, top, botton, left.
Ou usando a forma abreviada, padding: 5px 5px 4px 3px.
Pode-se também adicionar um unico valor a padding, dessa maneira ira considerar o espaçamento para todos os lados.
A padding não possui herança, dessa maneira se deseja mudar a fonte é necessario especificar dentro dela.

Margin

A margem controla o espaçamento entre dois boxes.

Ela frequentemente é usada em pixeis. Assim como nas propiedades anteriores, pode ser especificada inidivualmente ou dizendo a posição de cada margin.

Centralizando conteúdo

Para centralizar o box na pagina ou o seu conteudo, é necessario especificar os valores do left-margin e right-margin com o valor auto.

Inline/Block

A propiedade usada aqui é o display. O display possui 4 valores possiveis:
inline: Deixa os elementos em forma de linha, não podendo alterar as propiedades de altura e largura.
block: Deixa os elementos em um formato de bloco.
inline-block: Deixa os elementos em linha, sendo possivel alterar a largura e altura.
none: Deixa normal.

Bordas com imagens

A propiedade border-image aceita cologar imagens na borda. São três tipos de valores para essa propiedade. O url da image, o tamanho do fatiamento e o tipo de fatiamento.
Exemplo: border-image: url(/image/border.png) 30 repeat.
Para entendermos sobre fatiamento é necessario saber o tamanho da imagem. Obsver a imagem Os pontos laterais(laranja) são chamados de cantos, já os pontos internos(laranja claro) são as bordas. A propiedade round repede apenas as bordas não as laterais.
Ela possui um tamanho de 90x90, se a mesma é formada por três bolas em cada lateral, cada bola possui 27px. Dessa maneira podemos realizar o fatiamento de acordo com o tamanho da imagem e o fatiamento de uma unica bola. Para que apareça a bola inteira, colocamos o valor no tamanho de fatiamento = 27
e o tamanho da bordar utilizando border: 27. Caso queiramos 2 bolas, utilizamos 54(27+27).
Quanto aos tipos de fatiamento temos que:
stretch(alonga as bordas da imagem) repeat(repete as bordas da imagem) e round(da mesma maneira q o repeat porem se os ladrilhos nao se encaixam, eles vao se encaixar).
Usamos a imagem ao lado para fazer uma borda.

Usando o stretch: fatiamento tamanho 27 (uma unica bola)

Usando o repeat: fatiamento tamanho 27(uma unica bola)

Usando o round: fatiamento tamanho 54(duas bolas)

Box Shadow

Funciona de maneira similar da propiedade text-shadow. Na propiedade box-shadow podemos definir os valores do deslocamento vertical do box, a distancia e a sua cor. Ex.: box-shadow: 5px 5px 10px #FFFFF;

Arredondamento de cantos

border-radius

A propiedade border-radius é usada para arredondar os cantos de um box. Sendo especificado em pixels. Podemos especificar a cada canto usando: border-top-right-radius border-bottom-right-radius border-bottom-left-radius border-top-left-radius ou usando a forma abreviada(top, direita,inferior, inferior, esquerda) border-radius: 5px 10px 7px 1px; ou ainda especificar uma valor para todos os cantosborder-radius: 10px.

Formas Elipticas

Podemos criar complexos formatos especificando a distancia horizontal e vertical de cada arredondamento de canto. Exemplo: border-radius: 80px 50px ou ainda, cada canto com sua propiedade border-top-left-radius: 80px 50px
