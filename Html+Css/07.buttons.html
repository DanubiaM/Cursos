É formado da mesma maneira que os formularios anteriores apenas mudando o tipo de entrada para radio, com isso deve se alterar o valor de name(nome enviado ao servidor selecionado pelo usuario), value(valor que deve ser enviado para o servidor), checked( inicia a pagina com a opcao marcada).

Image Button

Para enviar imagems deve modificar o type do input para image, além disso vc pode delimitar o tamanho da imagem. No campo src você definir onde esta contido a imagem

Botão e controles escondidos

Você pode realizar combinações de texto e imagens dentro da abertura de um button. 'button 'img''. Por padrão se não definir o type do button ele é automaticamente definido como submit.
Comumente se usa o type hidden para alguma inclusao que não queira que o usuario veja. Como por exemplo o id de quando o formulario é enviado

Button=Submit x Input="submit"

Pode-se observer que os dois métodos tem a mesma funcionalidade, entretanto o button permite adicionar texto ou imagem entre sua tag, o que facilita sua edição e criação de butões com imagens. o button existe três tipos:

  • submit
  • disable
  • reset
Já o input pode conter diferentes formas além do submit, como: color, date, datetime, datetime-local, email, month, number, range, search, tel, time, url, week, text, file, hidden, image, month, radio, reset, password.

Label

Uma label pode ser utilizada para dois caminhos:

  1. Envolvendo o texto, a descrição e a entrada do formulario
  2. Separar o controle do formulario e qual formulario de controle é um rótulo
Dentro da tag label usamos o for, ele é usado para se associar com o id de uma entrada. Portanto o for deve conter o mesmo valor do id de uma input caso queira conectar ambas.

Genero:
Importante: a diferença do label é que se o input estiver conectado no id do label, ao clicar na palavra do label ja seleciona o input.

Fieldset

Para agrupar um formulario de controle utiliza-se a tag fieldset. Dentro dela utilize a tag legend para adicionar uma legenda. Para aparecer uma mensagem dentro de alguma entrada de texto, utilize dentro da tag placeholder="texto"
