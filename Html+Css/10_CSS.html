Aprendendo CSS

O CSS é o responsavel por estilizar a pagina. Para linkarmos ele à pagina html deve-se inserir um link no head, contendo dentro da tag link href, type(text/css), rel(stylesheet)

Style no head

A pagina pode ser estilizada de duas maneiras, criando um novo documento css e linkando ele como aprensentado anterioemente ou utilizando a tag style no head, dendo dela deve conter o type.

Selectores CSS

Existe diversas regras para selecionar uma tag e editar ela no CSS.

Seletor Universal * {}
Sera aplicado em todos os elementos da pagina
Seletor por tipo h1, h2, h3 {}
Aplicada a mudança nas tags
Seletor por classe .nomedaclasse {}
Aplica a mudança em todos os elementos que estão contidos naquela classe
Seletor por ID #nomedoid {}
Aplica a mudança em todos os elementos que estão com aquele ID
Seletor de crianças li>a{}
Aplica a mudança dentro da filha da tag
Seletor descendente p a {}
Tem como alvo quaisquer elementos que sente-se dentro de um elemento 'p', mesmo que existem outros elementos aninhados entre eles
Seletor irmão adjacente h1+p{}
Corresponde a um elemento que é o próximo irmão de outro
Seletor irmão geral h1~p{}
Corresponde a um elemento que é um irmão de outro, embora seja não tem que ser diretamente elemento precedente

Regra de Cascata CSS

Quando voce tem duas regras para se aplicar, ou seja dois tipos de você podera selecionar o que tem o procedenten anterior

Por exemplo, h1 e * h1 é mais especifico, então a regra se aplica a ele. i e p i é mais especifico então as alterações serao aplicadas a ele. Você pode usar o !important antes da propriedade para indicar que ele deve ser considerado o mais importante que as outras regras.
