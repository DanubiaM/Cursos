Tabelas

Tabelas são construidas utilizando a tag 'table' dentro utilize a tag 'tr' para colocar itens na linha dentro de tr coloque 'td' para colocar o dado da linha.
para variavas colunas utilize varias tr.

Matriz
a11 a12 a13
a21 a22 a23
a31 a32 a33

Para montar uma tabela com cabeçalho utilize a tag th e dentro dela coloque 'scope="linhaoucoluna"' ficando >> 'th scope='row'' para colocar cabeçalho na linha

A cada nova linha que desejo preencher uso um novo tr, para preencher a quantidade de colunas que desejo faço novos td ou th.
Para juntas duas ou mais colunas use 'colspan="n"' dentro da tag td ou th. Sendo n a quantidade de colunas que deseja juntar.
