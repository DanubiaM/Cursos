Links, como funcionam?

Existe diversas utilidades de links como para sair de um site para outro, de um site para o mesmo site em uma nova janela, abrir uma nova janela no navegador, abrir o email...
Os links são criados usando "a href".

  • Facebook acess
    Para colocar links de outros sites lembre-se de usar http:// na frente do link!
  • Info basic
    Caminhando em links do diretorio, se ja estiver no diretorio desejado utilize ./
  • email here
    Use mailto:email no link para acessar o email
  • Info Lists
    para abrir o link em outra pagina use "target="_blank""
  • Last topic
    Voce pode adicionar um id a cada tópico 'id="name"' e navegar nele utilizando-o como link, 'href="nameid"'
