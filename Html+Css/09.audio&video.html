Flash para video, animações e musicas

Esse tipo de trabalho atualmente é feito em Js, entretando pode se usar o adobe Flash para criar animações e inserir no site com o formato .swf atraves do SWFObject, usando script no head.
Esse scrip é feito no JavaScript. Dentro do primeiro script deve conter o local do script e no segundo o local onde estava o arquivo .swf

Video

Deve-se seguir 3 etapas para adicionar um video usando flash no site:

1º
Conversão do formato do video para o formato FLV.
2º
Ira precisar de um Player escrito em Flash. Grande parte são comprados.
3º
Incluir o video na pagina.Ira utilizar JS -> SWFObject
Entretando atravaes do HTML 5 é possivel também utilizando a tag video dentro dela deve conter algumas informações:
Tamanho
Especificar o tamanho com width e height
src
O caminho do video
poster
o caminho de uma imagem para ser o poster do video enquanto ele não for iniciado
controls
indica que o navegador pode usar seus própios controles para reprodução
autoplay
inicia o video automaticamente
loop
deixa o video em loop
preload
É chamado quando a pagina é iniciada e pode conter três valores none(o navegador não pode carregar o video antes que o usuario aperte o play) auto (o navageador pode carregar o video apos carregar) e metadata(o navegador pode coletar informações como tamanho, primeiro frame, track list e duração)

Outra maneira de adicionar video é usando a tag video dentro dela as especificações e entre o seu fechamento o src do video, podendo conter também um type (apresenta o formato do video antes para o navegador para verificar se é possivel carrega-lo) e o codecs(responsavel por codificar e decodificar o video, fazer sua transformação para outro formato, normalmente transformando em um video mais leve, os seus atributos depende do que deseja podendo ser H.264, DivX, MP4, AVI, MPEG-2, AVCHD, AAC, OGG, MKV )

Utilizando o atributo source você pode adicionar o video varias vezes com formatos diferente. Caso o autoplay esteja definido, o preload não ira funcionar

Audio

A adição dos audios no html é de forma parecida com o do de videos, apenas alterando a tag video para audio Os atributos são os mesmos.
