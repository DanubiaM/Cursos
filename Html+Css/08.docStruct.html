DOCTYPE

Todo documento que se deseja construir em html deve ser iniciado com um < !DOCTYPE html>, isso ajuda o navegador a indentificar a linguagem.

META

É um elemento dentro do head que contem as informações sobre sua pagina. Podendo ser <meta name=description,keywords,robots> < meta http-equiv= "author,pragma ou experies" >


pragma é provieniente do cach da pagina, usado para salvar o cach da pagina e diminuir o tempo na proxima visita à pagina.

ID atributo

Em html usa-se o atributo id em qualquer elemento ou tag. Isso ajuda a identificar cada elemento, facilitando assim sua edição no CSS ou adicionar interatividade com JS. Para isto, basta dentro da tag adicionar o nome id="identificação da tag" .

Class atributo

O atributo class dentro de uma tag serve parecido com o atributo id, entretando é utilizado para modificar varios elementos que estão contidos nessa classe. Dessa maneira a classe altera varias tags ou elementos, já o id apenas um unico, pois ele identifica apenas aquela tag.

Agrupamento de texto e elementos em um bloco: DIV

É possivel grupar varios itens juntos para criar um unico bloco, facilitando assim a edição no CSS. Para isso basta utilizar a tag div e dentro dela adicionar os elementos do bloco.
Isso é utilziado por exemplo para a criação do cabeçalho/menu do site, todos os elementos sao agrupados em uma div.

Iframe

O iframe é como uma pequena janela dentro da sua pagina onde pode ver outra pagina. Normalmente é utilziado para ver o google Maps
Os seus atributos são apenas src, height, width, scrolling, frameborder e seamless. Src, height e width já sabemos suas funcionalidader, o que muda são os atributos scrolling, frameborder e seamless.
Scrolling: Não é utilizado no HTML5 e sim no XML e HTML4. É utilziado para indicar se o iframe pode ter barra de rolagem ou não Aceitando que o usuario se mova dentro do iframe.
Frameborder: não é suportado no HTML5 e sim no HTML4 e XML. É usado para inidicar se tem borda, 0 quando não tem borda e 1 com borda.
Seamless: usado no HTML5 para indicar onde as barrar de rolagem não sao desejadas. Não é necessario ter valor, porem alguns autores utilizam.
