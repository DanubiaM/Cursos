Formulários, como funcionam?

Pode-se usar formularios para:

  • Adicionar Texto
  • Senha
  • Area de Texto
  • Marcar escolhar
  • Check Box
    • entre outros
A estrutura de um formulario é 'form', dentro dele deve conter as seguintes informações
'form= action"link do server" method="post or get" '
action
Dentro de action deve conter o URL da página no servidor que receberá o informações no formulário quando é submetido.
method: get or post
o metodo get deve ser espeficidado apos a action, é ideal para pequenos formularios e quando esta apenas recuperando dados de um servidor da web.
post -> são enviados no que são conhecidos como cabeçalhos http. É usado para arquivos longos, envio da informações confidenciais adiciona ou remove informações no banco de dados.

Formularios de entrada de dados

Para entrar com os dados em um formulario deve criar uma 'input'
'form action="link do server"' 'input type="tipo de entrada" name="username" size="tamanho" maxlength="maximo de caracteres"'
Dentro da entrada input adiciona o campo type onde define o tipo de dado no formulario de entrada
name é necessario para servidores saber o nome do dado que esta entrando, funciona como uma identificação do dado de entrada.
size é para alterar o tamanho do box, porem não é usado em codigos novos, sendo feito a alteração em CSS.
maxlength usado para definir o numero maximos de caracteres contidos no box de entrada.

Usuario:

Para senhas pode-se mudar o campo type para 'passoword'
